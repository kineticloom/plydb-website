PlyDB is a secure, unified access point for AI agents to query data in-place. From SQL databases like PostgreSQL and MySQL, flat files like CSV and Excel, or cloud sources like Google Sheets and S3 — PlyDB can query across them all with standard SQL, no data warehouse necessary.
